# API Documentation

این بخش شامل مستندات API خواهد بود.

## در حال توسعه

مستندات API در حال آماده‌سازی است.

## Endpoint های موجود

### Authentication

- `/api/auth/register/` - ثبت نام
- `/api/auth/login/` - ورود
- `/api/auth/logout/` - خروج
- `/api/auth/profile/` - پروفایل
- `/api/auth/password/change/` - تغییر رمز عبور
- `/api/auth/token/refresh/` - تازه‌سازی Token
- `/api/auth/token/verify/` - بررسی Token

### Health & Test

- `/api/health/` - بررسی وضعیت
- `/api/protected/` - Endpoint تست

برای اطلاعات بیشتر به [Authentication Documentation](../auth/) مراجعه کنید.
